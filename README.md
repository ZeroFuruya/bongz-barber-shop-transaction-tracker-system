# Bongz Barbershop Transaction Tracker

JavaFX desktop app for logging haircut transactions, barber commissions, and daily shop totals. Uses SQLite with foreign keys and role-based access (Owner / Manager).

## Run

Requires **JDK 17** and **Maven**.

```bash
mvn clean javafx:run
```

Main class: `bongz.barbershop.App`.
